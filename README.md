- 🎮 **Hard Mode**
- 🆘 **Help Screen**
- 🌗 **Light and Dark Mode**
- 📤 **Share Results**
- 📊 **View Statistics**

## 📦 Installation

Requires **Xcode 13+** and **iOS 15+**

1. Clone the repository:
   ```bash
   git clone https://github.com/Mridul-Learing-Unisa/Wordle.git

2. open Wordle.xcodeproj
3. Build and run on a simulator or your device.
